# About

個人開発で生成AIを使ったサービスを作るときに、PAY.JPのAPIを使ってサブスクリプションでの決済を実装するために作ったサンプルプロジェクトです。
以下の記事で詳しく解説しています。

https://qiita.com/gatchan0807/items/41c53c6dfe9401201f51

# Tech Stack

- Hono
- CloudFlare Workers & Pages
- Tailwind CSS
- Vite
- Gemini Nano in Chrome
- OpenAI GPT-4o-mini API
- PAY.JP API

# How to run

```txt
npm install
npm run dev
```

```txt
npm run deploy
```
